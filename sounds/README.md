# 音声ファイル

このフォルダには謎解きゲーム用の音声ファイルを配置します。

## 必要な音声ファイル

### 基本の音声効果
- `start.mp3` - 問題開始時の音（チャイム、ベル音など）
- `correct.mp3` - 正解時の音（ファンファーレ、成功音など）
- `wrong.mp3` - 不正解時の音（ブザー、残念音など）
- `hint.mp3` - ヒント表示時の音（ポップ音、チャイムなど）

## 音声ファイルの要件
- **形式**: MP3推奨（最も互換性が高い）
- **長さ**: 1-3秒程度（短く、効果的に）
- **音量**: 適度な音量（うるさすぎず、聞こえやすく）
- **ファイルサイズ**: なるべく小さく（50KB以下推奨）

## 無料音声素材サイト
- [効果音ラボ](https://soundeffect-lab.info/)
- [OtoLogic](https://otologic.jp/)
- [Freesound](https://freesound.org/)

## 音声ファイルの配置方法
1. 上記サイトから音声をダウンロード
2. ファイル名を変更（例：`correct.mp3`）
3. このフォルダに配置
4. GitHubにプッシュして更新
